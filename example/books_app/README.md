# Books App

A sample app that showcases common navigation scenarios:

1. Login/Logout/Sign-up Routing
1. Deep linking with path parameters
1. Deep linking with query parameters
1. Nested Routing with Tabs
1. Nested Routing with Modal Dialog
1. Skipping Stacks

For a description of each scenario, see the [Flutter Routing Scenarios v2](https://github.com/flutter/uxr/blob/6e4b3b5f5ba93036a4e767552bcf817a2bb0319b/nav2-usability/storyboards/%5BPublic%5D%20Flutter%20Navigator%20Scenarios%20Storyboards%20v2.pdf) document by the Flutter UX Research team.
